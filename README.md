## Description

A boilerplate for start up new Nestjs Project.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ docker compose up -d && npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
